Forbind til OneDrive

Første gang skal du indtaste dit Azure Client ID og stien til Excel-filen. Det gemmes i browseren og bruges automatisk fremover.

1Opret Azure App Registration
Gå til Azure Portal → App registrations → New registration

· Navn: Arbejdstid Dashboard
· Account types: Accounts in any organizational directory and personal Microsoft accounts
· Redirect URI: Web → https://k-stampe.github.io/arbejdstid-dashboard/

Klik Register og kopiér Application (client) ID.
2Tilføj API-rettigheder
Under din app: API permissions → Add a permission → Microsoft Graph → Delegated
Tilføj: Files.Read og Files.Read.All
Klik derefter Grant admin consent.
3Find fil-stien på OneDrive
Åbn onedrive.live.com, find filen, klik de tre prikker → Detaljer og notér stien.
Eksempel: /Dokumenter/Arbejdstidsskema.xlsx
Dine oplysninger gemmes kun lokalt i din browser. Ingen data sendes til tredjeparter.
