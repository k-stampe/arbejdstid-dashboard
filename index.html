Forbind til OneDrive

Opsaetning tager ca. 5 minutter og skal kun goeres een gang.

Trin 1 - Azure App Registration
Ga til Azure Portal og log ind med din private Microsoft-konto.
App registrations → New registration
• Navn: Arbejdstid Dashboard
• Account types: ...personal Microsoft accounts
• Redirect URI: Web → https://k-stampe.github.io/arbejdstid-dashboard/
Klik Register og kopieer Application (client) ID
Trin 2 - API-rettigheder
API permissions → Add → Microsoft Graph → Delegated
Tilfoj: Files.Read og Files.Read.All
Klik Grant admin consent

Data gemmes kun lokalt i din browser.
